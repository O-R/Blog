常用命令

apt

用户/用户组

  1. 查看所有用户 cat /etc/passwd

  2. 查看所有用户组 cat /etc/group

  3. 添加用户 sudo adduser or

  4. 将用户添加到 sudoers

    sudo vim /etc/sudoers

    修改文件如下：

    # User privilege specification
    root ALL=(ALL) ALL
    or ALL=(ALL) ALL

查看所有服务端口 cat /etc/services | less
