## 三者关系

> `镜像`从`仓库`里拉取到本地，成为`本地镜像`  
> `本地镜像`运行后是`容器`，运行前还是`镜像`

## 命令

| 命令 | 说明 |
| ---- | ---- |
| docker --version | 查看当前的版本 |
| docker ps | 查看当前正在运行的容器 |
| docker images | 查看本地所有镜像 |
| docker run | 运行一个容器 |
| docker pull | 用来拉取一个容器到本地 |
