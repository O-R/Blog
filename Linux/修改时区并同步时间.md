## 设置系统时间与网络时间同步

1.  安装ntpdate工具
    > sudo apt-get install ntpdate
    
1.  设置系统时间与网络时间同步
    > sudo ntpdate cn.pool.ntp.org
    
1.  将系统时间写入硬件时间
    > sudo hwclock --systohc //**非常重要，如果没有这一步的话，后面时间还是不准**
